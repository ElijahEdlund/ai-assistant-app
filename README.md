# AI Assistant App

Production-grade Expo React Native app with Supabase and OpenAI integration.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Copy `.env.example` to `.env` and fill in your credentials:
```bash
cp .env.example .env
```

3. Start the development server:
```bash
npm start
```

## Tech Stack

- Expo SDK 52
- Expo Router
- Tamagui (UI)
- Supabase (Backend)
- OpenAI (AI)
- TypeScript
- Zod (Validation)
